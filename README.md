# Big_project
красивый сайт. С небольшими курсами на тему "Глобальное потепление" и другие темы в конце курса пользователь будет проходить тест, чтобы закрепить знания. 
Также будут такие функции,возможности, как: 
   1)Регистрация;
   2)Возможность создания своих постов:
   3)Комментариев;
   4)Навигация по сайту;
   5)Поиск по сайту
   6)Администрация
   7)Раскрытие темы с помощью Тг-бота
